Proyecto prueba técnica Alegra

Formulario web que captura datos de comentarios sobre servicios de la empresa y se conecta con API de Gemini para hacer el análisis categorizando los mismos así como un resumen de estos.

--Tecnologías--
Front: HTML, CSS
Backend: JavaScript, Google Apps
IA para el análisis: Gemini 2.5 Flash
Dashboard: Looker Studio
Datos: Google Sheets

--Estructura--
index.html = Formulario
logica.js = Manejo de eventos y envío de información a Apps Script
Apps Script = Procesamiento de la información y conexión con API de Gemini
Informe de Feedback = Dashboard con los datos de la hoja de cálculo
Feedback = Hoja de cálculo donde se registran los datos del formulario


URL de la página: https://awhale-afk.github.io/Prueba-tecnica/
