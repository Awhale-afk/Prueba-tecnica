Proyecto prueba técnica Alegra

Formulario web que captura datos de comentarios sobre servicios de la empresa y se conecta con API de Gemini para hacer el análisis categorizando los mismos así como un resumen de estos.

Tecnologías:
Front: HTML, CSS
Backend: JavaScript, Google Apps
IA para el análisis: Gemini 2.5 Flash
Dashboard: Looker Studio
Datos: Google Sheets
